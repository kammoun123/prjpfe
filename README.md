# Gestion de Pièces Détachées

Système de gestion d'inventaire et de demandes de pièces détachées pour techniciens et magasiniers.

## Fonctionnalités
- Gestion du stock en temps réel.
- Système de demandes de pièces (Technicien -> Magasinier).
- Validation et transfert des demandes (Magasinier -> Admin).
- Rapports d'audit et historique des mouvements.

## Technologies
- **Backend**: Spring Boot (Java)
- **Frontend**: Angular
- **Base de données**: MySQL (Port 3307)
- **Serveur local**: XAMPP

## Configuration
1. Configurer MySQL sur le port **3307**.
2. Lancer le backend Spring Boot.
3. Lancer le frontend Angular avec `ng serve`.
